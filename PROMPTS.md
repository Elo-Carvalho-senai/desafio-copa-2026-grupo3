# PROMPTS UTILIZADOS DURANTE O DESENVOLVIMENTO

## Objetivo

Durante o desenvolvimento deste projeto, utilizamos Inteligência Artificial como ferramenta de apoio para pesquisa, organização, documentação e auxílio na programação.

Abaixo estão registrados os principais prompts utilizados, as respostas obtidas e as decisões tomadas pela equipe.

---

# Prompt 1

## Pergunta

Crie uma função JavaScript utilizando fetch e async/await para consumir uma API protegida por token JWT.

## Sugestão da IA

A IA sugeriu uma função assíncrona utilizando fetch, headers de autorização e tratamento básico de erros com try/catch.

## O que foi aproveitado

* Estrutura da função.
* Uso de async/await.
* Uso do cabeçalho Authorization.

## O que foi alterado

Foi adicionado tratamento para diferentes códigos de erro retornados pela API.

---

# Prompt 2

## Pergunta

Sugira uma estrutura de pastas para um projeto front-end utilizando HTML, CSS e JavaScript puro.

## Sugestão da IA

A IA sugeriu separar os arquivos em pastas específicas para CSS e JavaScript.

## O que foi aproveitado

Toda a organização da estrutura de diretórios.

## O que foi alterado

A equipe ajustou os nomes dos arquivos conforme a necessidade do projeto.

---

# Prompt 3

## Pergunta

Crie um layout moderno para exibir informações da Copa do Mundo utilizando apenas HTML e CSS.

## Sugestão da IA

A IA sugeriu a utilização de cards, tabelas e seções separadas para melhorar a experiência do usuário.

## O que foi aproveitado

* Organização visual.
* Estrutura dos cards.

## O que foi alterado

As cores, tipografia e disposição dos elementos foram personalizadas pela equipe.

---

# Prompt 4

## Pergunta

Como tratar erros de conexão em aplicações que consomem APIs?

## Sugestão da IA

A IA recomendou utilizar try/catch e verificar a propriedade response.ok antes de processar os dados.

## O que foi aproveitado

Toda a lógica de tratamento de erros.

## O que foi alterado

Foram adicionadas mensagens específicas para cada situação de erro.

---

# Conclusão

A Inteligência Artificial foi utilizada como ferramenta de apoio ao desenvolvimento, auxiliando na aprendizagem, na organização do projeto e na resolução de dúvidas técnicas.

Todas as decisões finais, adaptações e implementações foram realizadas pela equipe.
